# Portfolio

個人ポートフォリオサイト

## 使用技術

- **フレームワーク**: Nuxt 4 + TypeScript
- **パッケージマネージャー**: pnpm
- **UIコンポーネント**: shadcn-vue + reka-ui
- **スタイリング**: Tailwind CSS v4
- **3Dグラフィックス**: TresJS (Three.js)
- **コンテンツ管理**: @nuxt/content

## セットアップ

```bash
# 依存関係のインストール
pnpm install

# 開発サーバーの起動
pnpm dev
```

## コマンド

```bash
pnpm dev        # 開発サーバー
pnpm build      # 本番ビルド
pnpm generate   # 静的サイト生成
pnpm preview    # プレビュー
pnpm lint:fix   # Lint自動修正
pnpm textlint   # 文章チェック
```
